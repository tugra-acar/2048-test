# Assignment 3 Spring 26: 2048

This directory contains a simple Qt Widgets template app:

- a window title
- a title label in the UI
- a sample button
- a message box that appears when the button is clicked

## Build and run

```bash
cmake -S . -B build
cmake --build build
./build/2048
```

You can also use the verification script:

```bash
./scripts/verify_build.sh
```
